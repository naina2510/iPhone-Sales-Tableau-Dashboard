# 📊 iPhone Sales Dashboard

## 📌 Project Overview

This project is an interactive Tableau dashboard that analyzes iPhone sales data. It provides insights into sales performance, customer ratings, revenue, and product trends through interactive visualizations.

## 🎯 Objectives

- Analyze iPhone sales performance
- Compare sales across different regions
- Identify top-selling iPhone models
- Monitor customer ratings
- Support data-driven decision making

## 🛠️ Tools Used

- Tableau Desktop
- Microsoft Excel
- GitHub

## 📊 Dashboard Features

- Sales Overview
- Revenue Analysis
- Customer Rating Analysis
- Interactive Filters
- Product Performance Analysis

## 📂 Files Included

- iphone sales (share).twbx
- README.md

## 🚀 How to Use

1. Download the `.twbx` file.
2. Open it using Tableau Desktop or Tableau Public.
3. Explore the dashboard using the available filters.
